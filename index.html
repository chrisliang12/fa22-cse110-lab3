Product Goal

Develop a Pathfinding Visualizer to help users to learn the search pattern of different algorithms. Provide a playground for the users to play with different algorithms in the maze and visualize the results.

User Pain Points

In other similar products, the pain points usually are:

  • Ugly UI
  • Website is not responsive
  • Cannot stop the visualization
  • Does not provide multiple heuristics
  • Cannot customize the maze or start/end nodes

Supported Features

  • Well Designed UI
  • Support clear method during the visualization
  • Support different visualization speed
  • Provide different maze size
  • Provide maze presets
  • User need to be able to customize the maze or start/end nodes
  • Provide multiple heuristics
  • Support both 4 and 8 direction movements
  • Responsive website
